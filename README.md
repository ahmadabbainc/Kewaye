# Kewaye - Keke Napep Connect

A modern mobile and web application connecting Keke Napep owners with drivers to share daily shifts and maximize earnings.

## 🚀 Features

### For Owners
- Post daily work shifts
- Track driver performance
- Secure payment system

### For Drivers  
- Browse available shifts
- Accept jobs near you
- Build reputation through ratings

## 📱 Tech Stack

- **Framework**: Expo (React Native)
- **Navigation**: React Navigation
- **State Management**: Zustand
- **Styling**: NativeWind (Tailwind CSS)

## 🛠️ Setup

```bash
git clone https://github.com/ahmadabbainc/Kewaye.git
cd Kewaye
npm install
npm start
```
